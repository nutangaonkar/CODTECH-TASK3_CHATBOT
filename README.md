# CODTECH Internship – Task 3 (AI Chatbot with NLP)

This project implements a simple AI chatbot using Python and basic Natural Language Processing (NLP) concepts. The chatbot interacts with users through the command line and responds to predefined queries such as greetings, name, status, and exit commands.

### Technologies Used
Python, NLP (Natural Language Processing), NLTK

### Required Library Command
pip install nltk

### How to Run the Project
1. Install the required library using the above command.
2. Run the chatbot script using:
   python chatbot.py

### Sample Output
Chatbot: Hello! Type 'bye' to exit.  
You: hello  
Chatbot: Hello! How can I help you?  
You: what is your name  
Chatbot: I am an AI Chatbot built using NLP.  
You: bye  
Chatbot: Goodbye! Have a nice day.
